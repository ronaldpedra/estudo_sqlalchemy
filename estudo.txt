ORM - Object Relational Mapper - Mapeador de Objetos Relacionais

- As tabelas do banco de dados são representadas através de classes e os registros de cada tabela são representados como instâncias das classes correspondentes.

Banco de dados Relational       Programação Orientada a Objetos
Tabela                      ->  Classes
Coluna                      ->  Atributo
Registro                    ->  Objetos
